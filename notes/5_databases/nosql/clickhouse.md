#### **ClickHouse**

- **Что это:** Колоночная аналитическая база данных (OLAP). Не для транзакций, а для аналитики.
    
- **Как устроена:** Данные хранятся по колонкам, а не по строкам. Это позволяет сжимать их и очень быстро агрегировать (суммы, средние) по миллиардам строк.
    
- **Движки таблиц (MergeTree family):** Самый главный — `MergeTree`. Есть ещё `ReplicatedMergeTree`, `Distributed`.
    
- **Зачем:** Для аналитических отчётов, логов, метрик, где нужно считать много данных.
    
- **Команды:** SQL-like синтаксис, но с кучей функций для аналитики.
