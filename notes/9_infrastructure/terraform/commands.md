### Основной рабочий процесс (Workflow)

Стандартный цикл работы включает четыре основные команды:

1. **`terraform init`**: инициализация каталога. Инструмент скачивает нужные провайдеры и модули.
2. **`terraform plan`**: предварительный просмотр. Terraform сравнивает ваш код с реальным состоянием облака и показывает список изменений, которые он собирается внести.
3. **`terraform apply`**: применение изменений. Создает, изменяет или удаляет ресурсы, чтобы привести их к описанному в коде виду.
4. **`terraform destroy`**: полное удаление всей инфраструктуры, созданной в рамках текущего проекта
5. `-var-file=terraform.tfvars`: флаг для указания файлов переменных

| Команда                    | Что делает                                                                                                  |
| -------------------------- | ----------------------------------------------------------------------------------------------------------- |
| **`terraform init`**       | **Инициализация.** Скачивает провайдеры и настраивает Backend (хранилище State). Запускается первой.        |
| **`terraform validate`**   | **Проверка синтаксиса.** Проверяет код на ошибки, не подключаясь к облаку.                                  |
| **`terraform plan`**       | **Планирование.** Показывает, что будет создано, изменено или удалено. Самая безопасная команда для аудита. |
| **`terraform apply`**      | **Применение.** Создает реальные ресурсы. По умолчанию запрашивает подтверждение (`yes`).                   |
| **`terraform destroy`**    | **Уничтожение.** Полностью удаляет всю описанную инфраструктуру.                                            |
| **`terraform fmt`**        | **Форматирование.** Приводит все файлы в папке к единому стандарту оформления (красивые отступы).           |
| **`terraform show`**       | **Просмотр.** Выводит текущее состояние ресурсов из файла State в читаемом виде.                            |
| **`terraform state list`** | **Список ресурсов.** Показывает названия всех объектов, которыми сейчас управляет Terraform.                |
